💰 SimpleInvest – Smart Investment Guide & Robo-Advisor

SimpleInvest is a beginner-friendly investment dashboard with a built-in AI Investment Planner and Robo-Advisor.
It helps users understand investment options, assess risk, and receive personalized recommendations — all in a clean, modern UI.

Think of it as “Finance, but make it simple.” 😌📈

🌟 Features
📊 Interactive Dashboard

Explore different investment methods:

Stocks

Mutual Funds

Fixed Deposits

Cryptocurrency

Gold

Green Bonds

Smooth animations and modern card layout

🤖 Robo-Advisor

Step-by-step investment questionnaire

Inputs:

Monthly income

Risk appetite

Investment goal

Generates personalized investment recommendations

Displays:

Risk level

Expected returns

Best investment type

💬 AI Investment Planner (Chatbot)

ChatGPT-like UI with typing indicator

Answers finance questions related to:

Stocks

Budgeting

Risk & safety

Crypto

Simulated smart responses using JavaScript logic

📁 Portfolio Tracker (Mock)

Add investments from the dashboard

View invested amount & returns (simulated)

Clean empty-state UI for new users

🌙 Dark Mode

One-click dark mode toggle

Smooth theme switching using CSS variables

📱 Responsive Design

Fully responsive for desktop & mobile

Mobile sidebar toggle included

🛠️ Tech Stack

HTML5 – Structure

CSS3 – Styling, animations, dark mode

JavaScript (Vanilla) – Logic, state handling, chatbot, robo-advisor

Ionicons – Icons

Google Fonts (Outfit) – Typography

No frameworks. No libraries. Pure frontend magic ✨
